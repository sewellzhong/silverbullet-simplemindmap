# SilverBullet SimpleMindMap Plug

Readonly rendering for Obsidian Simple mind map `.smm.md` content in SilverBullet.

## Install

Use `Library: Install` in SilverBullet with this URI:

```text
https://github.com/sewellzhong/silverbullet-simplemindmap/blob/main/PLUG.md
```

Then run `Plugs: Update`.

If you prefer direct plug installation through the SilverBullet `CONFIG` page, use this exact value:

```lua
config.set {
  plugs = {
    "github:sewellzhong/silverbullet-simplemindmap/simplemindmap.plug.js"
  }
}
```

Do not keep an empty string or placeholder such as `<your-user>` in `plugs`; the plug manager will report `Could not resolve plug uri`.

## Usage

- Open a page generated by Obsidian Simple mind map.
- The `metadata` block is rendered as a readonly mind map when it contains valid `.smm.md` data.
- Pages tagged `simplemindmap` or using the Obsidian `metadata/svgdata/linkdata/textdata` structure open a readonly preview automatically.
- Run `SimpleMindMap: Preview Current Page` for a clean panel preview of the current page.

This first version does not edit or save mind maps from SilverBullet. Use Obsidian Simple mind map as the editor and SilverBullet as the viewer.

## Config

```lua
config.set {
  simplemindmap = {
    autoPreview = true,
    height = 720,
    fit = true
  }
}
```

## Build

```bash
bun install
bun run verify
```

The compiled output is `simplemindmap.plug.js`.
